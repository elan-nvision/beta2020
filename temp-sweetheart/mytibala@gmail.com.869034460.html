Epitaph

What was that? Whatever it was.
It was normal, or at least, seemed so.
But now it's gone and I can't remember
When it was that it became something worth
Remembering and rejoicing and relishing.
But, only a memory.

I was there. Not because you were there.
We were still there. Not for each other?
Yet we were there. Now, we're not.
But, it feels like we were for each other?

I thought you would be. Though first I didn't.
Now, I see what the second one didn't.
The first one was right. You aren't there.
But the third one is strong. I am still there.

The change was stark. A delta function.
But it is now gone. A mere flat turf now.
Seems like normal. Yet, it isn't. But, it is.
The autocorrelation function is still zero.
The system has lost the memory of its past.
The memory hasn't still left my system.
But you have.

I feel clean, refreshed and satisfied. 
Knowing I had regretted the loss of something that was not there.
It wasn't there. It hadn't felt so. So, it could not have been.
Now I just be.

An epitaph? 
An imagined tomb. 
An empty casket.
